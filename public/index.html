Troubleshooting

  • PC and ESP32 must be on the same WiFi network.
  • Use `canlogger.local`; if it fails, use the IP printed in Serial Monitor.
  • ESP32 works on 2.4 GHz WiFi networks.
  • If the logger is off, the dashboard shows demo-style fallback data.
